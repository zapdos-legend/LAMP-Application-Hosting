# LAMP-Application-Hosting
Hosted a LAMP stack application on AWS using EC2 and RDS with Linux, Apache, MySQL, and PHP for traditional web hosting.

## 🎯 Purpose

The purpose of this project is to deploy and host a traditional LAMP stack web application on AWS cloud infrastructure.

LAMP stands for:

* **Linux** – Operating System
* **Apache** – Web Server
* **MySQL** – Database
* **PHP** – Server-side Programming Language

This project demonstrates:

* Hosting PHP web applications on AWS
* Configuring Apache web server
* Connecting application with MySQL database
* Managing scalable cloud infrastructure
* Deploying traditional web applications securely

## 🧰 AWS Services Used

* **Amazon EC2** – Used for hosting the Linux server, Apache web server, and PHP application.
* **Amazon RDS** – Used for managing the MySQL relational database securely.
